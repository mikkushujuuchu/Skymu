# What is this folder?
Devin is the agent behind DeepWiki, a third-party service that maintains a comprehensive wiki of Skymu's source code that is used for developer reference and documentation.

# Does this mean Skymu is vibe-coded?
What a stupid idea. All Devin does is read the code in our repository and document it. The `wiki.json` file in this folder helps it correct inaccuracies during this process.

# I noticed some other (non-Skymu) repositories have `.claude` and `.gemini` folders. What are those for?
Those folders are generally for setting up agentic contributors, i.e. Claude makes commits to your repository automatically using its own account. Skymu does not allow agentic contributors on its repositories.
